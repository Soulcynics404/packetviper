# Features

## Implemented ✅
- Real-time packet capture via pnet
- Ethernet, ARP, IPv4, IPv6, ICMP, ICMPv6, TCP, UDP parsing
- HTTP, DNS, TLS (SNI), SSH basic detection
- Dashboard with live stats
- Packet inspection with hex dump
- Vim-style keybindings
- Multi-tab TUI interface
